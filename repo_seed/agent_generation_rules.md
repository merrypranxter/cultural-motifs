# AGENT GENERATION RULES

## Rule 1 — specificity over vibes

Always prefer a specific cultural, regional, formal, or material term over a vague aesthetic label.

Bad:
- ancient pattern
- tribal lines
- exotic ornament

Better:
- stepped fret pattern
- interlace knotwork
- vegetal arabesque
- lotus-and-papyrus border
- feathered serpent imagery

## Rule 2 — separate visual form from meaning

A motif can look similar across traditions while functioning differently.
Always separate:
- what it looks like
- where it appears
- what it may symbolize
- where interpretation becomes context-dependent

## Rule 3 — do not flatten adjacent traditions

When comparing neighboring traditions, name the distinctions explicitly.

Examples:
- Japanese decorative logic often differs from Chinese decorative logic in line behavior, composition, and emblematic use.
- Maya and Aztec/Mexica should not be treated as interchangeable.
- Islamic geometric systems, vegetal arabesque systems, and calligraphic ornament should be discussed separately before being linked.

## Rule 4 — anti-drift is mandatory

Every region or major subject should include some form of:
- what it is not
- common confusions
- common misuse patterns

## Rule 5 — no fake certainty

Use cautious phrasing when needed:
- often associated with
- commonly used in
- context-dependent
- not universal across all examples
- one recurring interpretation is

Avoid pretending every motif has one fixed universal meaning.

## Rule 6 — no tourist-shop language

Do not write in the tone of:
- a gift shop plaque
- a pseudo-mystical Pinterest post
- fantasy lore wiki sludge
- AI slop about “ancient energies”

## Rule 7 — machine usefulness matters

When relevant, include notes useful for:
- prompt construction
- visual retrieval
- shader logic
- generative composition systems
- tagging / app filtering

## Rule 8 — examples should be controlled

When giving examples, prefer:
- repeat structures
- composition tendencies
- motif families
- material tendencies
- formal comparisons

Avoid giant unstructured laundry lists.

## Rule 9 — headings should stay stable

Use consistent heading patterns so files are easy to compare across folders.

## Rule 10 — if in doubt, go narrower

Narrow and precise beats broad and fake.
