# What Is a Motif

## Purpose

This file defines what counts as a motif in this repository and establishes the basic unit of analysis.

---

## Definition

A **motif** is a recurring visual element or pattern that appears across multiple instances within a tradition or across traditions.

A motif is not just any mark or shape. It is a visual unit that:

- recurs recognizably
- has some degree of formal consistency
- functions within a larger visual system (border, field, panel, scene, garment, surface)

---

## What a motif is not

A motif is not:

- a single unique artwork
- a one-time design element with no repetition
- a purely verbal or narrative concept with no visual expression
- a vague aesthetic mood

---

## Motif dimensions

Motifs can be described along several dimensions:

### Visual form
What it looks like. Shape, line behavior, symmetry, contour, density, abstraction level.

### Compositional role
How it functions in a design. Border element, field filler, central emblem, corner ornament, scene participant, architectural accent.

### Scale
Where it appears in terms of size and placement. Icon, stamp, panel, garment band, wall surface, building facade.

### Material expression
What medium it is expressed in. Carved stone, painted ceramic, woven textile, inlaid metal, printed paper, tiled surface.

### Symbolic tendency
What it may represent. This varies by culture, period, and context. Symbolic readings should be kept proportionate to evidence.

---

## Motif families

Motifs can be grouped into families based on shared formal traits:

- **geometric** — grids, stars, frets, spirals, tessellations
- **floral / vegetal** — vines, lotus, palmette, arabesque, tree forms
- **faunal** — birds, serpents, lions, fish, hybrid beasts
- **figural** — deities, guardians, ancestors, humans in ritual
- **cosmological** — sun, moon, stars, cosmic axes, world-tree forms
- **architectural** — arches, columns, niches, gates, stepped platforms
- **border / frame** — meanders, guilloche, running patterns, edge containment
- **textile / surface** — repeat units, warp-weft logic, embroidery patterns

These families are not cultures. They are formal groupings used across the repo for classification and comparison.

---

## Why this definition matters

If the definition of "motif" is too broad, the repo becomes a dump of random visual stuff.
If it is too narrow, useful recurring patterns get excluded.

The working definition here is: a recognizable, recurring visual unit that functions within a tradition's design system.
